# MoodTrip Android MVP

MoodTrip adalah aplikasi MVP Smart Tourism untuk Mataram/Lombok. Aplikasi ini membantu pengguna mencari rekomendasi wisata, kuliner, dan landmark berdasarkan mood, tag, budget, dan keunikan lokal.

## Fitur yang Sudah Ada

- Tampilan home dengan konsep produk.
- Search berdasarkan keyword seperti `healing`, `pedas`, `Senggigi`, atau `hidden gem`.
- Chip filter mood/tag.
- Visual map MVP dengan pin lokasi.
- Daftar rekomendasi lokasi Mataram/Lombok.
- Detail lokasi berisi deskripsi, keunikan, harga, jam buka, rating, dan tag.
- Wishlist sederhana menggunakan SharedPreferences.
- Ringkasan prioritas fitur MoSCoW.

## Data Lokasi Awal

- Islamic Center NTB.
- Taman Sangkareang.
- Pantai Senggigi.
- Bukit Merese.
- Desa Sade.
- Warung Ayam Taliwang.
- Nasi Balap Puyung.
- Kafe Ampenan Heritage.

## Catatan MVP

Map pada versi ini masih berupa visual map custom, bukan Google Maps asli. Ini sengaja dibuat agar MVP bisa langsung dijalankan tanpa API key. Jika ingin naik ke versi berikutnya, map bisa diganti dengan Google Maps SDK atau Mapbox.
